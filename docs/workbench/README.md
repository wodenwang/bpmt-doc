# 文档工作台

本目录用于存放迁移过程中的草稿、代码解读笔记、章节校准记录和待办清单。

建议每个主题建立一个独立文件，例如：

```text
docs/workbench/quick-start.md
docs/workbench/database-init.md
docs/workbench/workflow.md
docs/workbench/dynamic-table.md
docs/workbench/script-functions.md
docs/workbench/permissions.md
```

每份工作台笔记建议包含：

- 目标章节
- 旧 GitBook 来源
- `bpmt-lite` 代码来源
- 当前确认结论
- 待验证问题
- 可直接进入正式文档的内容

工作台内容可以粗糙，但必须区分“已验证事实”和“待确认历史信息”。
