# wx 微信函数库

BPMT3.0 内置函数库 wx 提供了微信基础能力的封装，以下是所有的功能列表：

| 序号 | 函数 | 说明  |
|------|-------	-|-----|
| 0 | wx.mp('mpKey') | 获取mpKey所指定的mp函数集合  |
| 1 | wx.qy | 获取当前系统配置的企业号函数集合  |
| 2 | wx.agent | 获取当前系统配置的企业号下默认的agent函数集合  |
| 3 | wx.agent('agentKey') | 获取当前系统配置的企业号下agentKey指定agent函数集合  |
