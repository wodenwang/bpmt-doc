# math 数学操作
BPMT中内建的一套对数字的数学操作函数,比如汇总,计算平均值等  

## *math.sum* 汇总
    对一系列的数进行汇总计算

#### 参数API ####
| 序号 | 参数类型 | 说明  |
|------|-------	-|-----|
| 1 | List | 入参为数字的集合 |
| 返回值 | Number | 返回数字的集合汇总结果 |

#### 示例 1 :对数字集合(50,75,85,4,3,90,35,2)进行汇集计算
```groovy
return math.sum(50,75,85,4,3,90,35,2);
```
